# Delegation Feature

The delegation system enables automatic detection and handling of task delegation between agents.

## Documentation

- **[Delegation Detector](delegation-detector.md)** - Core delegation detection system
- **[Delegation Detector Hook](delegation-detector-hook.md)** - Hook implementation details
- **[Delegation Hook Summary](delegation-hook-summary.md)** - Summary of delegation hooks

## Overview

The delegation feature allows:
- Automatic detection of delegation patterns in agent output
- Routing tasks to appropriate specialized agents
- Tracking delegation chains and task completion
